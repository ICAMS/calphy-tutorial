# calphy-tutorial

Tutorial for calphy at MLIP fits 2026


Please first activate the environment:

```
cd
source load_calphy.sh
```

Then clone the tutorial repository:

```
git clone https://github.com/ICAMS/calphy-tutorial.git
```

Additionally to install the kernel,

```
python -m ipykernel install --user --name="calphy"
```

After this you can open the notebook `01_phase_transitions.ipynb` and choose the kernel `calphy`.
